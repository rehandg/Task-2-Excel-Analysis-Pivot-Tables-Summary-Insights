Task 2: Pivot Table Analysis & Insights
Objective

The objective of this task was to analyze a sales dataset using Excel Pivot Tables to summarize data, identify trends, and generate meaningful business insights. The task focuses on developing skills in data aggregation, filtering, visualization, and analytical storytelling.

Tools Used

Microsoft Excel

Pivot Tables and Pivot Charts

Slicers and Timeline for interactivity

Dataset

A structured sales dataset containing information such as:

Product

Region

Sales Method

Revenues

Expenses

Quantity

Date

The dataset was cleaned and converted into a table format before analysis.

Analysis Performed

Created pivot tables to summarize total revenue by product

Built a Region by Sales Method pivot to compare performance across regions

Added slicers and a date timeline for interactive filtering

Calculated Profit Margin using a derived column

Visualized results using pivot charts

Identified key trends and patterns from the pivot outputs

Key Insights

MacBook is the highest revenue-generating product.

North America is the top-performing region overall.

Online Store is the dominant sales method across regions.

Sales performance is balanced across products in APAC and EMEA.

Profit margins vary across regions, highlighting efficiency differences.

Deliverables

Pivot_Report.xlsx – Excel file with pivot tables, slicers, charts, and insights

Pivot_Report.pdf – Exported pivot analysis for sharing

Insights.txt / Insights.docx – Summarized analytical insights

Outcome

This task provided hands-on experience with pivot tables, data summarization, interactive dashboards, and insight reporting, closely aligning with real-world business intelligence workflows.
